# Giới thiệu

# Yêu cầu
1. [.Net6](https://dotnet.microsoft.com/en-us/download/dotnet/6.0) > 6.0.28
1. [Nodejs](https://nodejs.org/en/download) > 20.10.0

# Hướng dẫn debug

## Trước khi debug
- Đặt workspace đúng thư mục solution
- Mở terminal
- Vào thư mục frontend `cd frontend`
- Chạy lệnh `npm i`
